# Archival Family Tree – Al-Ruwaijh Lineage (شجرة الرويجح)

A modern Arabic-first interactive genealogy web application built to visualize and preserve the Al-Ruwaijh family tree in a clean, scalable, and interactive format.

---

## 🌳 Project Overview

This project transforms structured genealogical data into an interactive family tree visualization.

It is designed specifically for **Arabic lineage presentation**, focusing on clarity and scalability.

---

## ✨ Features

- Interactive expandable family tree
- Clean Arabic-first UI
- Optimized hierarchy rendering for large datasets
- Smooth animations

---

## 🧠 Data Rules

- Relationships are **father → sons only**
- No spouses or additional relationship types
- No inferred or fabricated data
- Core dataset located in `data/family.json`

---

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- D3.js (hierarchy rendering)
- Framer Motion (animations)
- Zustand (state management)
- Tailwind CSS

---

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Then open:

```
http://127.0.0.1:3000
```

---

## 📁 Structure

- `data/family.json` → main genealogy dataset
- `components/` → UI components
- `pages/` → routing
- `lib/` → tree logic
